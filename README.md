## onlineflix_microservice
this is node microservice test

### Development environment with docker
$ docker-compose pull  
$ docker-compose up --build -d

### end points
Base URL: **http://localhost:5000**
- GET /genres/:id
- GET /genres
- POST /genres
- PUT /genres
- DELETE /genres?id
- GET /movies/:id
- GET /movies
- POST /movies
- PUT /movies
- DELETE /movies?id
